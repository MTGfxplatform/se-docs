# Vungle

在 资产管理-三方账号管理-投放平台 页面，点击右上角添加账户，选择Vungle进入账户添加页面

![](<../../../.gitbook/assets/image (174).png>)

Vungle平台需要输入的字段为：

* 账户：您的登录邮箱，标识该账户
* API key：调用接口的必需字段
* Secret Key：调用接口的必需字段
* Refresh Token：调用接口的必需字段
