# Applovin

## 添加账户

在 资产管理-三方账号管理-投放平台 页面，点击右上角添加账户，选择Applovin进入账户添加页面

![](<../../../.gitbook/assets/image (88).png>)

Applovin平台需要输入的字段为：

* 账户
* Report Key
* Client Id
* Client Secret

对应在Applovin后台各字段的位置：

Account-Basic Info

* Email->账户

![](<../../../.gitbook/assets/image (18).png>)

Account-Keys页面：

* Report Key

![](<../../../.gitbook/assets/image (17).png>)

Client ID和Client Secret通过Account>>OAuth Apps页面获取
