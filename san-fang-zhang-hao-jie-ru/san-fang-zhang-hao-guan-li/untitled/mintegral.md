# Mintegral

## 添加账户

在 资产管理-三方账号管理-投放平台 页面，点击右上角添加账户，选择Mintegral进入账户添加页面

![](<../../../.gitbook/assets/image (85).png>)

Mintegral平台需要输入的字段为：

* username：调用接口的必需字段
* API key：调用接口的必需字段

对应在Mintegral后台各字段的位置：账号管理-用户管理

* 用户名称->user\_name
* Api key->API key

![Mintegral后台](<../../../.gitbook/assets/image (2).png>)
